为 Debian 和 Ubuntu 发布签名 APT 元数据，软件包下载重定向到可信的上游 .deb 文件，不重新分发二进制包。

查看软件包
install.sh 
curl -fsSL __BASE_URL__/key.asc | sudo gpg --dearmor -o /usr/share/keyrings/lyk-ai-apt.gpg
echo "deb [signed-by=/usr/share/keyrings/lyk-ai-apt.gpg] __BASE_URL__ __SUITE__ __COMPONENT__" | sudo tee /etc/apt/sources.list.d/lyk-ai.list
sudo apt update

发布模型

APT 元数据由本仓库生成并签名；软件包文件不经本站托管，下载请求会重定向到对应的上游发布地址。

校验边界

客户端通过仓库公钥验证索引签名，并由 APT 校验包文件大小与哈希；建议使用 signed-by 绑定密钥。
